## 🎯 Какво ще направим утре (10-15 минути)

Ще качим CRM-а онлайн на **2 места**:
1. **Railway** - за backend (API + база данни)
2. **Vercel** - за frontend (React приложението)

И двете са **безплатни** за малки проекти.

---

## 📋 Какво ще ти трябва

### Преди да започнем:
- [ ] GitHub акаунт (ако нямаш, ще направим утре)
- [ ] Email адрес
- [ ] 10-15 минути време

### Няма нужда от:
- ❌ Кредитна карта (100% безплатно за начало)
- ❌ Сървър или хостинг
- ❌ Техническа експертиза
- ❌ Инсталация на нищо

## 🗺️ Пътят, който ще следваме

```
Код на компютъра
        ↓

        ↓

        ↓

        ↓
    🎉 Готов CRM с публичен линк!
```

---

## 🚂 Част 1: Railway (Backend + Database)

### Какво е Railway?
Представи си го като "облак" където пускаш код и той автоматично:
- Прави база данни
- Пуска твоя backend
- Дава ти линк

### Какво ще направим:
1. Влизаме в railway.app
2. Свързваме GitHub
3. Избираме нашия проект
4. Казваме му "пусни backend папката"
5. Добавяме PostgreSQL база данни (1 клик)
6. Готово!

### Какво получаваме:
- URL на backend: `https://something-production.up.railway.app`
- База данни която работи
- Всичко без да настройваме нищо

---

## 🚀 Част 2: Vercel (Frontend)

### Какво е Vercel?
Място където пускаш React приложения. Специализирани за това.

### Какво ще направим:
1. Влизаме в vercel.com
2. Свързваме GitHub
3. Избираме проекта
4. Казваме му "frontend папката"
5. Добавяме 1 променлива (линка към Railway)
6. Deploy!

### Какво получаваме:
- URL на CRM: `https://your-crm.vercel.app`
- Автоматично HTTPS (сигурен)
- Бърз като светкавица

## 📝 Стъпките (най-простата версия)

### GitHub (ако нямаш акаунт)
```
1. Отиди на github.com
2. Sign up
3. Потвърди email
4. Готово
```

### Railway Deploy (5 мин)
```
1. Отиди на railway.app
2. "Login with GitHub"
3. "New Project"
4. "Deploy from GitHub repo"
5. Избери твоя CRM проект
6. Root directory: "backend"
7. Add PostgreSQL (бутон "+ New" → Database → PostgreSQL)
8. Изчакай deploy (1-2 мин)
9. Копирай URL-а (Settings → Domains)
```

### Vercel Deploy (3 мин)
```
1. Отиди на vercel.com
2. "Sign up with GitHub"
3. "Add New..." → "Project"
4. Import твоя CRM проект
5. Root directory: "frontend"
6. Environment Variables → Add:
   REACT_APP_API_URL = (твоя Railway URL)/api
7. Deploy
8. Готово!
```
